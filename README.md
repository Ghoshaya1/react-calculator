# React Calculator

## Description
Auto-generated project by AI Agent Platform

## Tech Stack
- React 18\n- Vite\n- JavaScript/ES6+\n- CSS3\n- ESLint\n- Prettier

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)\n- Clear and delete functionality\n- Decimal number support\n- Keyboard input support\n- Responsive design for mobile and desktop\n- Error handling for invalid operations\n- Display formatting for large numbers

## Getting Started

### Prerequisites
Install dependencies as per tech stack above.

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install  # or yarn install, pip install -r requirements.txt, etc.
```

### Running
```bash
npm start  # or appropriate command for your stack
```
